
# Platforms

The structure contains platforms that APIMatic CodeGen can generate SDKs and Docs in.

## Enumeration

`Platforms`

## Fields

| Name | Description |
|  --- | --- |
| `CSNETSTANDARDLIB` | .NET Standard Library |
| `JAVAECLIPSEJRELIB` | JAVA |
| `PHPGENERICLIBV2` | PHP |
| `PYTHONGENERICLIB` | Python |
| `RUBYGENERICLIB` | Ruby |
| `TSGENERICLIB` | Typescript |
| `GOGENERICLIB` | Go |

## Example

```
CS_NET_STANDARD_LIB
```
